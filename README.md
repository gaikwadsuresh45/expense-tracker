# Expense Tracker

A full-stack web application to track daily expenses.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask REST API
- Database: SQLite
- Version Control: Git & GitHub

## Features
- Add expenses with category (Food, Travel, Books, Other)
- Delete expenses
- Real database storage
- Category-wise tracking

## How to Run

### Backend
pip install flask flask-cors
python app.py

### Frontend
Open dasboard.html with Live Server

## Project Structure
- login.html - Login page
- dasboard.html - Main dashboard
- app.py - Flask backend
- expenses.db - SQLite database
